Hello all,

You can murder me if this is totally off topic and/or unwanted, but I figured I'd run it up 
the flag pole.

I've created a domain name to share via freedns.afraid.org for anybody who wants to 
setup and self host their own gemini server but doesn't want to / isn't able to register 
their own domain.  I made it short and sweet so subdomains look more like a domain in 
its own right.  

I've gone for tb.je, TB being "Text Based" (muy listo), JE being, well Jersey, one of the 
few ccTLDs with short domains left in 2020.

If you're interested you can register via https://tb.je or direct from the freedns registry 
(https://freedns.afraid.org/domain/registry/)

For those with dynamic IPs, freedns can also handle updating the record automatically 
for you : https://freedns.afraid.org/dynamic/

Any questions, I'm happy to take them !

-Kevin

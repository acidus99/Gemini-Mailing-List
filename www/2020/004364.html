Hi everyone,
I wrote a messageboard for Gemini. It uses client certs to register and authenticate users, and runs as a CGI script. Users can also append to posts they have created so post length isn't limited by the max URL length. By default it's structured like a textboard, but it can be configured pretty much any way you like.

Right now it's only been tested with Jetforce but it should work with any Gemini server that provides the same CGI variables.

There is a live version at gemini://[illegaldrugs.net/cgi-bin/motm](http://illegaldrugs.net/cgi-bin/motm/boards).
Source code and instructions can be found at https://tildegit.org/sose/motm.

Hope it proves useful for some.

- sose

If you already have a gopher server, then it should be easy to set up a 
gemini server on the same box. I would recommend solderpunk's Unsinkable 
Molly Brown. It is written in Go, so it is pretty easy to build. If you 
are unable to build it yourself, Go statically links its binaries, so 
you could perhaps ask someone to email you the binary.

https://tildegit.org/solderpunk/molly-brown

Feel free to email me directly if you have trouble.

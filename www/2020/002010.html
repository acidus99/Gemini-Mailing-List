Hi!

I have updated  gemini browser included in  tinmop: a distraction-free
pleroma client.

This  version  (0.1.0)  improves  the gemini  navigation,  now  it  is
possible to  open an arbitrary  address (using command line  switch or
using the Terminal interface) and i  have added a "go back" command to
browse backward the addresses the user already visited.

Of course it  is still possible to open gemini  links contained in the
body of a message posted on the social network.

Tinmop  is  free  software  (GPLv3+ licensed),  more  information  are
available here:

https://www.autistici.org/interzona/tinmop.html

Bye!
C.

changes on its gemini browser, i do  not want to spam the mailing list

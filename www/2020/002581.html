Greetings Geminauts!

After putting some more work into my Prolog Gemini server I am happy to 
announce that I now have a working capsule running over at 
gemini://gemini.wtf.

This capsule also provides simple Gemtext hosting capabilities to 
everyone who's interested. The intention is to give anyone access to it 
tilde-style and the Prolog app serves whatever is inside the `public` 
directory in your home. Currently it only correctly serves .gmi files 
but if anyone is interested in making use of this and wants to serve 
content other than gemtext I'm happy to add more mime/extension 
mappings. Note that every user gets full SSH access to the box so you 
could set up cronjobs etc. It defaults to bash as shell for new users 
but again on request I can make other stuff available too. Same goes for 
scripting languages. There's the Ubuntu defaults like (ba)sh, perl, etc, 
but also SWI Prolog. Although it is missing other popular things like 
Python right now.

If anyone wants to get hooked up with an account all I need is your 
desired username and your public SSH key.

Have a nice day/evening/night,

~Paul Kosel

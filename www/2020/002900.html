I wrote an anonymous textboard for the Gemini protocol. Written in Rust
using Tokio to provide high performance asynchronous and multithreaded
processing. PostgreSQL is used for a database.

Source: https://git.kiwifarms.net/CrunkLord420/gemboard

Test Instance: gemini://matrix.kiwifarms.net/

Future plans include more robust spam detection, post throttling and
optional/multiple TLS certificate support for use behind a reverse proxy
to facilitate dual clearnet and tor onion domains.

You manage Gemini servers? Despite the fact that the protocol is still
quite experimental, you want to do a serious job, which means
*monitoring*? You use a monitoring system that can use external plugin
using the Nagios API (like Nagios itself, Icinga and many others)?

Then Manisha is here to help. It it a monitoring plugin for Gemini
servers. It follows the usual rules of the Nagios API and will allow
you to be informed of what happends to your Gemini server(s).

Hey guys,

I wrote a little ssg for gemini for fun and games (and to practice Go).
I had a inspiration from kiln, but I added a few personal touches.

A few features worth mentioning:
* Go templates
* Date sourcing from filename or the page content.
* Atom feeds
* A init command to startup a new gemlog.

My idea here is to make it as simple as possible to start a gemlog.
There's default templates which (should) just work.

(The default index template should be compatible with the subscription
format solderpunk just announced, btw)

I'm using it now to generate my website.

The code needs some cleanups, but it's working.

If you do check it out, feel free to point me at stupid code or bugs :)

Enjoy,

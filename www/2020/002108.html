Hi everyone

I'm pleased to announce a new release of GemiNaut 0.8.6, a particularly 
user friendly Gemini client for Windows.

The main change in 0.8.6 release is that GemiNaut now supports Gopher, 
so you can now browse Gopher space too. Due to the lack of accessible 
clients, there have perhaps not been so many Windows users exploring the 
gopher holes for many years. I've been discovering gopher space for 
myself in testing GemiNaut and whilst it is a slower medium than Gemini, 
there is quite a lot of thoughtful writing to discover. Needless to say 
gopher pages look really nice in GemiNaut which smooths over the visual 
gaps providing a common experience of gemini, gopher maps and text files.

Key changes and updates:

  - support gopher, with same theming engine as for gemini
  - search in gopher (e.g. Veronica)
  - updated faster "Fabric" theme generated directly by GemiNaut
  - scroll bars added when needed to each preformatted area as the 
window size decreases
  - auto linking of URLs in gopher text files
  - use latest gemget 0.1.3
  - other minor UI improvements and bug fixes

I'd like to put out a word of appreciation to makeworld for the gemget 
utility and the prologic guys for the go-gopher library, the latest 
versions of both of which have been incorporated.

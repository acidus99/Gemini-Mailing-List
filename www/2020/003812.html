I've written a program which aggregates feeds formatted according to the
"Subscribing to Gemini pages" specification. Extremely similar
functionality to Emilis' gmi-feed-aggregator, just implemented differently.

https://tildegit.org/emilis/gmi-feed-aggregator
gemini://gemini.circumlunar.space/docs/companion/subscription.gmi

You can find more details here:
gemini://calcuode.com/gemlog/gmisub.gmi

And an aggregate generated by it here:
gemini://calcuode.com/gmisub-aggregate.gmi

I'm planning to have the above page aggregate all the feeds I can find so
it can be a reference for anyone setting up their own more focused
aggregator. If you know of any feeds I'd appreciate a link!

There are some pages I've found that almost follow the specification but
have a slightly different date format or non-descriptive title, so I'll
probably try contact a few of those people to see if they would be up for
changing.

>
> Minor correction:
>
> Titan is a PUT protocol that can send something to a server.  Dioscuri is a true POST protocol: you send something to the server, and then you get either a response or an URL pointing to the true response (thanks to the magic of MIME-type message/external-body).  Dioscuri isn't really intended as a file-upload protocol, though it could be used that way, but as an application protocol for requests that won't fit into an URL.  It is in no way intended for use with forms, but with specialized clients.

I wrote a new Gemini browser.

It supports client certificates, input, TOFU certificates, non UTF-8
encodings, binary downloads, non Gemini links, has bookmarks, history
etc. with vim-style and cursor-y shortcuts.

I run it on my Mac, Raspberry Pi Zero W, Raspberry Pi 4 (Ubuntu).

Code and binaries is over at https://github.com/a-h/min if you want to
give it a go.

Thanks!

Hi all,

SecretShop, a Gemini server written in Go, is now available for
download. You can grab the source from my Git repo[1]. The latest
version is actually 1.0.1 as I released a quick fix after.

Technically this has existed for a while, but I finally got around to
handling certificates and officially releasing a 1.0 so I figured I'd
make a formal announcement.

I've also packaged it up on OBS so any OpenSuse users can download an
RPM for it from my home repo[2]. Fedora and other RPM users will be
coming next, followed up by Debian whenever I learn how to generated deb
files. The repo might be a bit behind and only have 1.0.0 right now but
that should change.

I've also set up my own capsule at saintnet.tech[3], both as a personal
site and as a demonstration of SecretShop. There ain't much there yet
but I'm working on it. It'll probably end up being a mismatch of
personal posts, metal reviews, and random rants about tech, philosphy,
and weird religions.

Cheers,
Steve

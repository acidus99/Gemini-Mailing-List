> I’m curious about how authors structure their workflows in geminispace. My own is somewhat of a work in progress 😆
> 
> I write my posts in vim, manually add an entry to my archive gemini feed and then run a bash script that updates my capsule index page and my atom feed. It’s a bit clunky, and I’m pondering how to make it less cumbersome.
> 
> What’s your workflow like?

I just use vim through ssh to write directly on the server.

The gemlog index is dynamic and just lists the files in the folder.

The downside is that the name of the file is used as post name, I may update the generated index to fetch the first line of posts and use that as link name, not sure.
I decided against using the file creation date automatically as post date, because I want to be able to write dateless pages, which will be updated in the future, while a gemlog post with a date is supposed to be a one shot.

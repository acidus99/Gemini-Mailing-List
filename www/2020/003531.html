gmitohtml, a Gemini to HTML proxy, is now available. It can run as a daemon,
which allows browsing Gemini pages using a standard web browser. It can also run
as a document conversion tool, reading from standard input.

gmitohtml is available on Android devices as Xenia.

https://gitlab.com/tslocum/gmitohtml
https://gitlab.com/tslocum/xenia

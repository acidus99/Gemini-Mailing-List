<https://gemini.circumlunar.space/docs/specification.html> the text
"Valid values for the lang parameter are comma-separated lists of one
or more language tags as defined in RFC4646". RFC 4646 has been
replaced eleven years ago. I suggest to change the specification and,
instead of using the actual RFC number, use the "BCP" reference which
is stable. "BCP 47" pointed to RFC 4646 before and now to the current
one, 5646.

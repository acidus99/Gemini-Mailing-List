Hello Geminauts,

Today I released Amfora, my client for Gemini. It aims to be a great looking,
featureful browser... all in the terminal. It is similar to Bombadillo, but
has colors and stylings, *tabbed browsing*, and brings in a lot more deps.
Sloum, if you're reading this, thanks so much for all the work you've done
with Bombadillo.

I aim to add a lot more features to it over the coming weeks, like bookmarks,
downloading, header navigation / TOC, client certs, and feed subscribing.
Let me know what you think!

Github: https://github.com/makeworld-the-better-one/amfora
Binaries: https://github.com/makeworld-the-better-one/amfora/releases

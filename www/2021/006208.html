Onion sites often have clearnet counterparts that send an HTTP header to 
inform the user agent that a hidden site exists; that way, the user can 
switch from connecting to the clearnet site through an exit node to 
connecting to the hidden service without any exit nodes involved.

Some way to inform the user agent to switch to the hidden version when 
available would be appreciated, since exit nodes are the weakest point 
of the Tor network. Without some way to automatically "upgrade" to the 
hidden version, users might end up accidentally using the clearnet 
version of the site.

After lots of development, astronaut 0.1.0 is finally here!

https://sr.ht/~adnano/astronaut

astronaut is a Gemini browser for the terminal. Its features include:

- Vim-inspired interface with tabs and mouse support
- Commands and configurable keybindings
- Link follow mode for quick navigation
- Jump mode for jumping to certain headings
- Search for text in page

astronaut has come a long way in terms of performance and stability.
There is still a lot of work to be done to polish some features like
downloading or bookmarking, though.

Enjoy!

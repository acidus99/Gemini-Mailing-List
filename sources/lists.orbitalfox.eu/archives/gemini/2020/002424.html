Hi all!

I wanted to be able to visit both the web and geminispace from a single
program so I wrote a small userscript for qutebrowser. It downloads the
gemini content, converts it to Markdown and then to HTML using Pandoc.
Yes, it's an ugly hack and not all of gemini is implemented, but it
works acceptably well for browsing gemini content. 

The code is based on gcat so many thanks to it's authors!

Get it here:

https://git.sr.ht/~sotirisp/qute-gemini

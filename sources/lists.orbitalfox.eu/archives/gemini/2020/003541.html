Thanks for the feedback! I'll work on improving the documentation.

The default index template includes links to directories by default.
I wasn't sure if this would be a desired feature, and I can remove it.

The default template also only includes links for any Gemini files in a
directory. I could expose the non-Gemini files to the template so you
could list them manually if you desire.

You can always create your own templates if the default ones don't work
well. The default templates can be found at
https://git.sr.ht/~adnano/kiln/tree/master/templates/

I've been working on a Gemini client for the terminal:

https://sr.ht/~adnano/astronaut

It has tabs, mouse support, and support for custom user styles.

It also passes all of the torture tests (except 0050, which causes
unicode artifacts for some reason).

There's still a lot of work to be done, including client certificate
management, bookmarks, custom key bindings, and some documentation.

Enjoy!

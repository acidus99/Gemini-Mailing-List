Hello all,

Lagrange is a cross-platform desktop GUI client for browsing Geminispace. It is written in C and uses an SDL2-based custom UI toolkit with no dependencies on existing GUI frameworks. The app is designed to be lightweight, fast, and nice to use.

=> gemini://skyjake.fi/lagrange/

After five months of coding, version 1.0 is here.

Version numbers can be pretty arbitrary but the switch from major version zero to one has special significance. In the case of this project it denotes a shift of gears, as the initial rapid iteration phase ends and things should proceed in a calmer and more orderly fashion. In other words, version 1.0 is the first stable release.

While version 1.0 should be feature-rich enough to be comfortable to use, I have a long list of improvement ideas for the existing features and for adding new ones. Now is a time to take stock and reprioritize — after a while it should become clear which areas are lacking and what can wait until later.

Version 1.0 introduces a couple of bookmark related features. For example, you can use a Gemini page as a source of remote bookmarks. You can read more about this here:

=> gemini://skyjake.fi/gemlog/2020-12_lagrange-1.0.gmi

--jaakko

Hi everyone

I'm pleased to announce a new release of GemiNaut, a user friendly 
graphical client for Gemini on MS Windows.

The main change in this 0.8.7 release is that GemiNaut now opens images 
directly within the client, and linked files can be downloaded directly 
without use of a web proxy. This makes for a more fluid and natural 
experience browsing geminispace.
There are also some minor usability improvements, for example, menus can 
be accessed using keyboard shortcuts, File->New to open a new window, 
and minor theme improvements.

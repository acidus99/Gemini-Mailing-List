No, clients are free to format it, wrap it, apply fonts and colours, etc.
If it was preformatted, then links wouldn't work, for example.
There's a difference between this:

=> gemini://gus.guru/ GUS

and this:

```
=> gemini://gus.guru/ GUS
```

In the second one, a client will not form a link, it will just show the raw text.
It should also show the text in a monospace font, whereas it might apply some
other font for the first example.

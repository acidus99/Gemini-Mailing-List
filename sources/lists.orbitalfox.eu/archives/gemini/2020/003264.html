So it is a bit strained, but the idea is that you can describe gemtext 
in the following terms: there are various line types, each line type 
except Text and Preformatted is introduced by a 1-3 char prefix; any 
line which doesn't start with one of the prefixes is considered to be 
Text; [rules about toggling to preformatted mode]. Then this change 
would delete the "except Text and Preformatted" from this summary.

It's even more strained because alt-text doesn't fit with this "list of 
lines" interpretation of a gemtext.

Anyway, this change would be pretty close to backwards-compatible. The 
only real downside is that it adds two new magic prefixes.

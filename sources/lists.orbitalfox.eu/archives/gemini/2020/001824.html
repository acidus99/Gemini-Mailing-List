HTML defines the rel attribute for the <link> element: the rel attribute specifies the relationship between the current document and the linked resource.

text/gemini can broadly achieve the same functionality by the expedient of consistently annotating its link lines:

=> gemini://example.org/favicon.png <gemini://example.org/favicon.png> 🆁🅴🅻 ICON IMAGE/PNG
=> gemini://example.org/atom.xml <gemini://example.org/atom.xml> 🆁🅴🅻 ALTERNATE APPLICATION/ATOM+XML

There is a pleathora of rel values to choose from:

http://microformats.org/wiki/existing-rel-values <http://microformats.org/wiki/existing-rel-values>

Any interest to flesh this out in more details?

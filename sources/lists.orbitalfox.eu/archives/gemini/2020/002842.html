Hi All

Just to let you know there is a new release out this week of GemiNaut, 
the user friendly Gemini client for Windows.

The main changes in this release are:

1. Simple website browsing - view most simple websites directly in 
GemiNaut, and only launch your system browser when you have to. Uses 
Html2Gmi utility to convert web pages to GMI files, or you can also 
connect to an Http-to-gemini proxy such as Duckling. A good majority of 
sites linked from Gemini are within the so called Small Web (document 
not application centric sites), and these will generally display well 
enough to get a good sense of the content.

2. Signed application binary and new installer. Some folk reported 
getting false positives from anti-virus utilities, incorrectly marking 
GemiNaut as malware. The official installer and application binary are 
now signed. So it should produce fewer scary warnings when you install 
and run it.

3. Minor UI improvements, including better readability with increased 
line spacing, scroll bars on preformatted areas only added on hover, can 
display text/html pages on Gemini sites.

Further details and links to installer and github repo on project 
website and gemsite

https://www.marmaladefoo.com/pages/geminaut

gemini://gemini.marmaladefoo.com/geminaut/

Any feedback and bug reports welcomed.

Best Wishes

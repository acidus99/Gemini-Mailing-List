I feel obliged to tell you that my wiki engine was, in fact, the first application to implement gemtext outside gemini :)

=> https://github.com/bouncepaw/mycorrhiza

Fun fact: my gemtext implementation kinda contradicts the specification so it can better fit the whole wiki idea. It supports transclusion (a feature not easily implemented in an actual geminisphere), handles link urls differently (so it is easier to link other wiki pages, I call such links wiki links) and empty lines are handler roughly (they are ignored (the spec recommends rendering them with vertical space)).

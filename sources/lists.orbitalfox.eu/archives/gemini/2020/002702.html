I noticed that Molly Brown supports Gemini-syntax files named “.mollyhead” so one can have text before a directory listing. So, I added support for .mollyhead files to the .sublime-syntax file I have.

I don’t know of any other similar file extensions that might have Gemini text in them. If you know of a (dot)file that does, let me know about it so I can add it to the list.

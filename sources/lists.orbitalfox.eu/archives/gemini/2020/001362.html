Hi, everybody. I was reluctant to use the [ANN] tag in the subject
because I don't have a new Gemini server to announce *yet*, but I
wanted to let everybody know that I've started packaging some Gemini
software for Slackware as Slackbuilds.

I currently have Slackbuilds for the following clients available on my
website, and I've also submitted them to slackbuilds.org. 

- bollux
- castor

Once they've been approved, Slackware users will be able to install by
hand or using tools like sbopkg or slpkg.

Tomorrow I'll take a stab at creating Slackbuilds for some of the
server software, starting with gemserve.

PS: The release tarballs will need to be renamed after downloading them
from Sourcehut because their filenames are "${VERSION}.tar.gz" rather
than "${NAME}-${VERSION}.tar.gz". I don't know if tooks like sbopkg
account for this or if I'll have to email Drew DeVault and report this
as a bug. :)

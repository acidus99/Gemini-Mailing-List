Hi all,

A few months ago I released the first public build of my iOS browser for Gemini called Rocketeer. Following feedback, I’ve completely rewritten the UI layer. Rocketeer now supports edge-swipe gestures for back and forwards navigation, context menu previews, text selection, a table of contents, and displaying images! Additionally a whole bunch of bugs have been fixed, notably redirects not being followed in some circumstances.

There are still a bunch of features yet to come, including multi-window support on iPad, bookmarks, home page customization, and a Mac version!

If you’d previously installed it, you can update from the TestFlight app (and you should soon; the current build will expire in just 7 days). If you’d like to try it out, follow the instructions here: https://testflight.apple.com/join/LAs1URxM <https://testflight.apple.com/join/LAs1URxM>.

In related news, TestFlight recently gained support for automatic app updates, so you won’t have to risk the app expiring on you. As such, this should be the last time I email the list until the full-on App Store release.

As before, if you run into any issues or have other feedback, I want to hear about it! You can email me at this address or get in touch on the fediverse (I’m @shadowfacts at social.shadowfacts.net).

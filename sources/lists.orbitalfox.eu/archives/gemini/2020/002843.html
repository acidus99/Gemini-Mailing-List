I've created a collection of Gemini software that can be installed with
the Nix package manager. Most of these packages are in the main Nix
repositiory but there are collected here for convience. Since the Gemini
protocol is still experimental, it may be better to first submit packages
to a staging area like this.

=> https://git.sr.ht/~ehmry/gemini-flake

I don't have time to package every client and server, but I'm happy to
accept patches, and I can also review packages in the main repository.

Also, for anyone using NixOS who hasn't noticed, there is a configuration
module for molly-brown.
=> gemini://gemini.spam.works/~emery/nixos-hosting.gmi

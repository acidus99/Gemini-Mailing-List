Hello everyone,

Seduced by the ideas behind gemini protocol, I also found gemtext so much relaxing in the meantime.
Thus, I decided to move my writings to a new capsule:

=> gemini://ybad.name ArraKISS

It is indeed very pleasing to write without inline links or other markup, even the simplest ones such as **bold text** one can find in markdown. I think it helps wondering how to write better (at least, it does for me).

Actually, I publish both on http and gemini, but focus on gemini using a stupid gemtext to html converter. It follows gemini specification (i.e. a line is a <p></p>). I call gemtext2html in a Makefile to convert modified gmi files to html, create an index of all pages, publish an atom feed of my gemlog...

=> https://dev.ybad.name/gemtext2html/

Mostly, I write in french, but sometimes in english.

Also, I enjoy contributing to vger, the gemini server wrote by @solene, that I also use.

=> https://tildegit.org/solene/vger

I must say I feel very happy with gemini and enjoy reading other capsule with the very well thought lagrange client.

Thank you for making this possible.

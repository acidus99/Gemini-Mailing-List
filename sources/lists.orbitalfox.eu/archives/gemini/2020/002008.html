Welcome to tilde.pink! I hope you brought your shades. Your account is
now activated with the username evn. SSH login is via key only.

We do not offer web services. We use geomyidae instead as our
gopher server. It uses .gph files, instead of gophermaps. Default is
index.gph. A public_gopher folder has been created for you with a sample
index.gph file. We've also added a gemini server. Gemini is a gopher
like protocol that is currently in development. It is being designed to
fix some of the drawbacks of gopher, but without having all the issues
of html. You can experiment with gemini by editing and adding things to
your public_gemini folder. There is some initial documentation linked
on the main page.

Please note, the server is in its early stages and very bare bones.
Please let me know if something is missing that you are used to using
in a shell. Also, this is a NetBSD installation. There are many things
different from your normal linux system including your default shell
which is /bin/sh instead of bash. Bash is available if you would like
to change it.

If you are interested in talking to other ~pink users or need help,
please join us via irc on #pink hosted on the irc network tilde.chat

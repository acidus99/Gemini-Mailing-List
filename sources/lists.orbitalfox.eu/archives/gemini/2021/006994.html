Hello everyone,

I wanted to announce the release of stargazer 0.5.0
https://sr.ht/~zethra/stargazer/

stargazer is a concurrent Gemini server using async io with no runtime 
dependencies (if compiled with musl libc). It main features are: 
multi-host, route configuration, CGI, SCGI, and automatic cert generation.

This release adds some new features including: lang and charset 
configuration, finer cert controls, and FreeBSD support.

Check out the release notes here for more details:
https://git.sr.ht/~zethra/stargazer/refs/0.5.0

Cheers,
